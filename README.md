# Laba2-Fadeev-IDB-25-07  
Отчет по лабе

В ходе лабораторной работы была создана программа «История браузера» на язвке C++. Структура данных: Двусвязный список, поскольку необходимо ходить назад/вперед через указатели и быстро удалять ветки истории без сдвига памяти.

Функционал:  
Навигация Назад / Вперед.  
Поиск по домену.  
Вывод истории таблицей.  
Сохранение в файл (Base64).  
Установка закладок.  
Очистка истории.  

Итог: Всё работает, утечек памяти нет, граничные случаи обработаны.
